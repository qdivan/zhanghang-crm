## 当前业务能力
1. 客户开发
- 未成单线索录入、跟进、提醒、放弃、转客户
- 已存在客户自动建议并可复用，避免重复建客户

2. 客户列表与客户档案
- 已成交客户独立维护
- 客户时间线记录成单前后重要事项
- 客户事项支持提醒、办结、结果回写
- 香港公司模板支持年度事项提醒，且同周期重复套用会被拦截

3. 收费明细
- 以会计收费单视角展示应收、实收、余额、到期提醒
- 支持催收、收款、续费、提前终止
- 公司名可直接展开往来账

4. 到账核对
- 按入账账户与时间范围查看流水
- 支持老板、管理员、部门经理核对到账

5. 常用资料 / 公开资料
- 常用模板、通讯录、扩展模块
- 内部资料与公开资料隔离
- 公开资料可直接给官网复用

6. 管理与权限
- 角色：老板 / 管理员 / 部门经理 / 会计
- 支持直属经理关系与范围控制
- 支持临时只读授权
- 支持 LDAP 设置与同步入口
- 本地账号支持按 IP 的错误登录锁定

## 角色边界
1. 老板
- 可看全部业务数据
- 不可管理管理员账号

2. 管理员
- 可管理本地账号、授权、LDAP 设置
- 可看全局管理数据

3. 部门经理
- 只能看直属下属范围内的客户、收费、到账核对
- 不可越权查看非下属数据

4. 会计
- 只能看自己负责客户及收费数据
- 默认不可进到账核对
- 有只读授权时可额外看指定模块
